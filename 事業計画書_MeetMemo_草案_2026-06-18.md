## 1. エグゼクティブサマリー

### 事業の一言要約

> 「会議の録音をAIが自動で議事録化し、タスクを担当者に通知する」SaaS——**MeetMemo**

### 解決する課題

ビジネスパーソンは議事録作成に年平均 **320時間** を費やし、67%が「業務負担」と感じている（キヤノンMJ調査）。さらに会議後のタスク管理は既存ツールでは手動が前提となっており、「誰が・何を・いつまでにやるか」が曖昧なまま放置されるという構造課題が残っている。

### 提供価値

録音するだけで、AIが①文字起こし→②議事録整形→③タスク自動抽出→④担当者へのSlack/メール通知まで **一気通貫で実行**。会議を「成果に変わるアクション」に繋げる唯一のSaaS。

### ターゲット

従業員 **50名以下の中小企業**。国内357万社のうち、AI議事録ツールの導入率はわずか **1.4%**（DXマガジン調査）と未開拓市場が広大。

### ビジネスモデル

月額課金 **3,000円/ユーザー**。フリートライアル14日間でプロダクト価値を体験させてから有料転換するPLG（Product-Led Growth）モデルを採用。

### 3年間の財務ハイライト

|--|------|------|------|
| ユーザー数（年末） | 100名 | 400名 | 1,200名 |
| MRR（年末） | 300万円 | 1,200万円 | 3,600万円 |
| 年間売上高 | 約1,400万円 | 約8,000万円 | 約2.9億円 |
| 営業損益 | **▲1,300万円** | **+3,100万円** | **+1.9億円** |

## 2. 課題と解決策

### 2-1. 課題の実態

**課題①：議事録作成に膨大な時間がかかる**

- ビジネスパーソンが議事録作成に費やす時間は **年平均320時間**（8時間換算で40営業日分）
- 1時間の会議に対し、事後の整理・清書・共有に **2〜3.5時間** が発生
- AI導入後は **15〜30分** に短縮可能（2025年AI議事録導入企業調査）
- 67%が「業務負担」と感じながら、AI活用ツールの導入率は **わずか1.4%**

**課題②：会議後のタスク管理が属人的かつ曖昧**

- 議事録は作成されても「誰が・何を・いつまでにやるか」が口頭確認止まりになりがち
- タスクの抜け漏れが発生し、後からの「言った・言わない」問題の温床になる
- 既存の議事録ツールは「文字起こし・要約」止まりで、タスク管理フローとは断絶している

**課題③：中小企業には使える選択肢がない**

- 高機能ツール（Otolioなど）は月額25,000円〜と中小企業には高価
- 安価なツール（Nottaなど）はタスク通知機能がない
- 海外ツール（tl;dv、Otter.aiなど）は日本語UIや日本の商習慣への対応が不十分

### 2-2. 解決策

**MeetMemoが実現するフロー（録音開始ボタン1つで全自動）:**

```
会議録音 → [AI] 文字起こし（日本語特化）
        → [AI] 議事録整形（テンプレート自動適用）
        → [AI] タスク・決定事項の自動抽出
        → [自動] 担当者へSlack/メール通知
        → [管理画面] タスク進捗の一覧確認
```

| 機能 | 詳細 |
|------|------|
| **AI文字起こし** | 日本語特化LLMによる高精度文字起こし（話者識別付き） |
| **議事録自動生成** | 「決定事項」「課題」「タスク」の構造化フォーマットに自動整形 |
| **タスク自動抽出** | 「○○さんが△△を□□までに行う」を自然言語から自動検出 |
| **担当者への自動通知** | Slack・ChatWork・メールに担当者別タスクを会議終了後即時通知 |
| **タスク管理画面** | 全社のタスク進捗をダッシュボードで一覧管理・リマインド送信 |

## 3. 市場規模（TAM・SAM・SOM）

### グローバル・国内の市場環境

| 市場 | 規模（2025年推定） | 成長率（CAGR） |
|------|-------------------|---------------|
| グローバル音声認識市場 | **約2.8兆円**（$183.9億） | **22.98%**（→2030年 $517.2億） |
| グローバルAI SaaS市場全体 | **約20兆円**（$1,317億） | **38.3%**（→2030年 $6,731億） |
| 国内AI音声認識市場 | **約3,500億円** | **34.8%**（2034年まで継続） |

### TAM・SAM・SOM（試算）

```
TAM（グローバルAI会議支援SaaS市場）
  音声認識市場の約15%が会議用途と推定
  └→ 約 4,100億円

  ▼

SAM（国内中小企業向けAI議事録SaaS市場）
  対象: 従業員10名以上の中小企業 約70万社のうち
        DX意欲があり導入可能な企業 30% ≒ 21万社
  想定単価: 月額5ユーザー × 3,000円 × 12ヶ月 = 18万円/社/年
  └→ 約 1,260億円（21万社 × 18万円）

  ▼

SOM（3年以内に獲得可能な現実的市場）
  SAMの0.1〜0.2%（約1,200〜2,400社）
  └→ 約 20〜40億円
```

| 区分 | 規模 | 根拠 |
|------|------|------|
| **TAM** | 約4,100億円 | 国内外AI会議支援SaaS市場全体 |
| **SAM** | 約1,260億円 | 国内中小企業（DX意欲あり21万社）× 年間単価18万円 |
| **SOM** | 約20〜40億円 | SAMの0.1〜0.2%、3年以内に獲得可能なシェア |

### 参入タイミングの根拠

- AI議事録ツール国内導入率 **1.4%** は巨大な「まだ使っていない層」の存在を意味する
- 人手不足倒産が2024年に過去最多となり「省力化SaaS」への投資優先度が急上昇
- IT導入補助金2026の継続でSaaS導入の初期障壁が大幅に下がっている

## 4. ビジネスモデル

### 料金プラン

| プラン | 月額 | 対象 | 主な機能 |
|--------|------|------|---------|
| **フリートライアル** | 0円（14日間） | 全企業 | 全機能利用可能（14日限定） |
| **スタンダード** | **3,000円/ユーザー** | 従業員〜50名の中小企業 | AI議事録・タスク抽出・Slack通知・管理画面 |
| **チーム（予定）** | 12,000円/月（〜5ユーザー） | 小規模チーム向け | スタンダードと同機能（チームパック） |

> **価格設計の根拠**: 月額3,000円/ユーザーは「クレジットカードで即断できる価格帯」。5ユーザーのチームで月15,000円。議事録作成に年320時間かかっている場合の時給換算（2,000円/h）では年64万円のコスト = **ROI 40倍以上**。

### 収益構造とユニットエコノミクス

| 指標 | 目標値 | 算出根拠 |
|------|--------|---------|
| **ARPU（月次）** | 3,000円/ユーザー | 単一プライシング |
| **月次チャーンレート** | 3%以下 | SMB SaaSの健全目標値（業界平均 1〜7%） |
| **LTV** | 約10万円/ユーザー | ARPU ÷ Churn = 3,000 ÷ 0.03 |
| **CAC（顧客獲得コスト）** | 約3万円/ユーザー（目標） | LTV/CAC = 3倍以上を維持 |
| **CAC回収期間** | 約10〜12ヶ月 | 業界ベンチマーク「12ヶ月以内」を目標 |

### グロース戦略（PLG + SMB向けインバウンド）

```
認知 → フリートライアル → 有料転換 → 継続 → 口コミ紹介
 ↑                                              |
 └── コンテンツ/SEO / 比較サイト / 補助金経由 ←──┘
```

1. **PLG（Product-Led Growth）**: 14日フリートライアルで「録音→タスク通知」を体験させ、有料転換を促進
2. **補助金チャネル**: IT導入補助金2026の対象ツール登録。補助金申請経路での集客と、実質50%オフでの提供
3. **インバウンドSEO**: 「議事録 効率化」「AI議事録 中小企業」等のキーワードでコンテンツ集客
4. **パートナー**: 中小企業支援に関わる税理士・社労士・ITコーディネーターとの紹介代理店連携

## 5. 競合優位性

### 主要競合との比較

| 評価軸 | **MeetMemo** | Notta | Otter.ai | LINE WORKS AiNote |
|--------|:----------:|:-----:|:--------:|:-----------------:|
| 日本語認識精度 | ◎ | ◎（98.86%） | △（英語主体） | ◎（90%以上） |
| タスク自動抽出 | **◎** | ✕ | △ | ✕ |
| 担当者への自動通知 | **◎** | ✕ | ✕ | ✕ |
| Slack/チャット連携 | **◎** | ✕ | △ | △ |
| 中小企業向け価格 | ◎ | ◎（月1,185円〜） | ○（$16.99〜/月） | ○（198円/時間〜） |
| 日本語UI・サポート | ◎ | ◎ | △ | ◎ |
| セルフオンボーディング | ◎ | ◎ | ○ | △ |

### 各競合の概要と弱み

**Notta**
- 強み: 日本語認識精度98.86%・操作が直感的・個人向け低価格
- 弱み: タスク抽出・担当者通知機能なし。文字起こし止まりで業務フロー連携がない

**Otter.ai**
- 強み: 英語圏での認知度・精度が高い。月$16.99〜とリーズナブル
- 弱み: 日本語対応が不十分（認識精度・UIともに）。タスク通知機能なし。日本語サポートなし

**LINE WORKS AiNote**
- 強み: LINE WORKSユーザーとの統合・従量課金（198円/時間〜）・セキュリティ実績
- 弱み: タスク自動抽出・担当者通知機能がない。LINE WORKSを使っていない企業への訴求が弱い

### MeetMemoが"勝てる3軸"

```
① タスク通知 ── 会議後のアクションを自動化する唯一のツール
                 競合は「議事録を作る」だけ。MeetMemoは「会議を実行に変える」

② 中小企業最適価格 ── Otolioより1/8以下の価格でタスク通知まで実現
                       補助金対応で実質半額〜無料での提供も可能

③ 日本語×業務フロー完結 ── 海外ツールが苦手な日本語×商習慣に最適化
                             録音→議事録→通知→タスク管理を1ツールで完結
```

## 付記：リスクと対応策

| リスク | 影響度 | 対応策 |
|--------|--------|--------|
| Zoom/Teams等が議事録+タスク通知を標準搭載 | 高 | プラットフォーム非依存の設計・独自AIモデルの強化・データ資産（会議ナレッジ）の構築 |
